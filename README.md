#Trivia Quiz App

A clean, fast, and accessible trivia quiz application built with React and tanstack-start. Perfect for learning, quick-play sessions, and embedding into larger projects.

##About

This Trivia Quiz App is a single-page application that serves randomized trivia questions, scores user performance, and provides category/difficulty filtering. It uses React for the UI and tanstack-start as the starter scaffold (to rapidly set up TanStack libraries and recommended conventions).

Use cases: standalone quiz site, learning tool, embed inside portfolios, or as a frontend reference for TanStack Query/Router patterns.

##Tech Stack

-React (functional components + hooks)

-tanstack-start (project scaffold / recommended setup)

-TanStack Query (recommended for data fetching & caching)

-React Router or TanStack Router (depending on scaffold)

-CSS / Tailwind / Styled Components (pick per project configuration)
